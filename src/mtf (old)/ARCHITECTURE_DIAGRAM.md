## 🔄 Поток данных

```
OHLCV Data → Features Module → Context Builder → Context Data
     ↓              ↓              ↓
OHLCV Data → Features Module → Triggers Builder → Triggers Data
     ↓              ↓              ↓
Context Data + Triggers Data → Consensus Builder → Consensus Data
     ↓              ↓              ↓
All Data → MTF Pipeline → Final Signals → Database
```
