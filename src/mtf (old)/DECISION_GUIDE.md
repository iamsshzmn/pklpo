# MTF Decision Guide - Руководство по принятию торговых решений

## 🎯 Обзор системы

MTF Decision Maker - это система для принятия торговых решений на основе многотаймфреймового анализа. Система анализирует данные с 9 таймфреймов и предоставляет читаемые рекомендации.

## 📊 Структура анализа

### 1. Контекстный анализ (1M, 1W, 1D, 4H, 1H, 15m)
- **Назначение**: Определение общего направления рынка
- **Показатели**: trend score, валидность, режим рынка
- **Вес в решении**: 40-50%

### 2. Триггерный анализ (15m, 5m, 1m)
- **Назначение**: Точный тайминг входа
- **Показатели**: вероятности разворота, ускорение, микро-фильтры
- **Вес в решении**: 30%

### 3. Consensus анализ
- **Назначение**: Финальное решение по горизонтам
- **Показатели**: side (-1/0/+1), score (0-1), горизонт
- **Вес в решении**: 20-30%

## 📈 Интерпретация результатов

### Уровни уверенности
- 🟢 **HIGH** (≥0.7): Сильный сигнал, рекомендуется вход
- 🟡 **MEDIUM** (0.4-0.7): Умеренный сигнал, осторожный вход
- 🔴 **LOW** (<0.4): Слабый сигнал, не рекомендуется вход

### Типы сигналов
- 🟢 **LONG**: Покупка (бычий тренд)
- 🔴 **SHORT**: Продажа (медвежий тренд)
- ⚪ **FLAT**: Ожидание (боковик)

### Уровни риска
- **Низкий**: ≤1 конфликт в контексте
- **Средний**: 2 конфликта в контексте
- **Высокий**: >2 конфликтов в контексте

## 🎯 Стратегии торговли

### Swing Trading (1-7 дней)
**Критерии входа:**
- Consensus score > 0.6
- Валидный контекст на 1D
- Минимальные конфликты между таймфреймами
- Подтверждение на 4H

**Управление рисками:**
- Стоп-лосс: 2-3% от цены входа
- Тейк-профит: 6-9% от цены входа
- Выход при изменении consensus

### Intraday Trading (1-4 часа)
**Критерии входа:**
- Consensus score > 0.5
- Положительное ускорение на 5m
- Прохождение микро-фильтра ликвидности
- Подтверждение на 15m

**Управление рисками:**
- Стоп-лосс: 1-2% от цены входа
- Тейк-профит: 3-5% от цены входа
- Выход при развороте ускорения

## 🔧 Настройка системы

### Пороги уверенности
```python
CONFIDENCE_THRESHOLDS = {
    ConfidenceLevel.HIGH: 0.7,    # Сильный сигнал
    ConfidenceLevel.MEDIUM: 0.4,  # Умеренный сигнал
    ConfidenceLevel.LOW: 0.0      # Слабый сигнал
}
```

### Критерии принятия решений
```python
# Сильный сигнал
if consensus_score > 0.7 and context_conflicts <= 1:
    signal = consensus_side
    confidence = HIGH

# Умеренный сигнал
elif consensus_score > 0.5 and context_conflicts <= 2:
    signal = consensus_side
    confidence = MEDIUM

# Слабый сигнал
else:
    signal = FLAT
    confidence = LOW
```

## 📋 Чек-лист для принятия решений

### Перед входом в позицию
- [ ] Consensus score > 0.5
- [ ] ≤2 конфликтов в контексте
- [ ] Подтверждение на ключевых таймфреймах
- [ ] Прохождение микро-фильтров
- [ ] Соответствие горизонту торговли

### Управление позицией
- [ ] Установлен стоп-лосс
- [ ] Определен тейк-профит
- [ ] Мониторинг consensus изменений
- [ ] Отслеживание конфликтов в контексте

### Выход из позиции
- [ ] Достижение тейк-профита
- [ ] Срабатывание стоп-лосса
- [ ] Изменение consensus сигнала
- [ ] Появление новых конфликтов

## 🚨 Важные предупреждения

1. **Не торгуйте против сильного тренда** - consensus должен совпадать с контекстом
2. **Учитывайте горизонт торговли** - внутридневные сигналы не подходят для swing
3. **Следите за конфликтами** - много конфликтов = высокий риск
4. **Используйте управление рисками** - всегда устанавливайте стоп-лосс
5. **Не переторговывайте** - дождитесь качественных сигналов
