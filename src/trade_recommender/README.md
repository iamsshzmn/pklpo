# 📊 Система торговых рекомендаций

Этот модуль автоматически анализирует торговые сигналы и создаёт готовые рекомендации для торговли криптовалютами. Он берёт данные о сигналах, проверяет их качество и рассчитывает все необходимые параметры для открытия позиции.

## 🎯 Что делает система

Представьте, что у вас есть робот-советник, который:

1. **Анализирует сигналы** - проверяет, стоит ли доверять торговому сигналу
2. **Рассчитывает параметры** - определяет цену входа, стоп-лосс и тейк-профит
3. **Управляет рисками** - автоматически рассчитывает размер позиции и риск
4. **Даёт готовые рекомендации** - выдаёт полную инструкцию для торговли

## 🔍 Как это работает

### Шаг 1: Проверка качества сигнала
Система проверяет каждый сигнал по нескольким критериям:
- **Вероятность успеха** должна быть не менее 60%
- **Преимущество** должно быть больше 1%
- **Уверенность** должна быть не менее 30%

### Шаг 2: Расчёт параметров позиции
Если сигнал прошёл проверку, система рассчитывает:

**Для покупки (LONG):**
- **Цена входа**: текущая цена актива
- **Стоп-лосс**: цена входа минус расстояние (на основе волатильности)
- **Тейк-профит**: цена входа плюс удвоенное расстояние до стопа

**Для продажи (SHORT):**
- **Цена входа**: текущая цена актива  
- **Стоп-лосс**: цена входа плюс расстояние
- **Тейк-профит**: цена входа минус удвоенное расстояние до стопа

### Шаг 3: Управление рисками
Система автоматически:
- Ограничивает риск до 2% от баланса
- Рассчитывает размер позиции на основе риска
- Определяет необходимое плечо

## 🔧 Настройки системы

### Основные параметры риска:
- **Баланс**: 20 USDT (по умолчанию)
- **Риск на сделку**: 2% от баланса
- **Множитель стопа**: 1.5 (на основе волатильности)
- **Соотношение риск/прибыль**: 1:2

### Критерии качества сигналов:
- **Минимальная вероятность успеха**: 60%
- **Минимальное преимущество**: 1%
- **Минимальная уверенность**: 30%

## 📈 Алгоритм расчётов

### 1. Расчёт стоп-лосса
```
расстояние_стопа = волатильность × 1.5
стоп_цена = цена_входа - расстояние_стопа (для покупки)
стоп_цена = цена_входа + расстояние_стопа (для продажи)
```

### 2. Расчёт риска
```
риск_usdt = баланс × 0.02 = 0.4 USDT
```

### 3. Расчёт размера позиции
```
количество = риск_usdt / |цена_входа - стоп_цена|
```

### 4. Расчёт тейк-профита
```
тейк_профит = цена_входа + 2×(цена_входа - стоп) (для покупки)
тейк_профит = цена_входа - 2×(цена_входа - стоп) (для продажи)
```

### 5. Расчёт плеча
```
стоимость_позиции = количество × цена_входа
плечо = стоимость_позиции / баланс
```

## 💡 Полезные советы

1. **Начните с тестового режима** - используйте `dry_run=True` для проверки без сохранения
2. **Мониторьте качество сигналов** - регулярно проверяйте статистику отклонённых сигналов
3. **Настройте параметры под себя** - измените баланс и риск в соответствии с вашей стратегией
4. **Ведите журнал** - сохраняйте все рекомендации для последующего анализа

## 🔄 Жизненный цикл рекомендации

1. **Получение сигнала** → Система находит сигнал в базе данных
2. **Проверка качества** → Анализирует вероятность успеха и преимущество
3. **Расчёт параметров** → Определяет цены входа, стопа и тейка
4. **Создание рекомендации** → Формирует готовую инструкцию для торговли
5. **Сохранение в базу** → Записывает результат для последующего использования
6. **Исполнение позиции** → При необходимости создаёт запись об открытой позиции
