### 📊 Материализованные представления

**Файл:** `src/db/migrate_materialized_views.py`

**Созданные представления:**
1. **mv_symbol_stats** - статистика по символам и таймфреймам
2. **mv_latest_prices** - последние цены для каждого символа
3. **mv_daily_aggregation** - дневная агрегация OHLCV данных
4. **mv_volatility** - метрики волатильности по дням
5. **mv_top_symbols** - топ активных символов за последние 7 дней
6. **mv_data_quality** - метрики качества данных

**Автоматизация:**
- ✅ Функция `refresh_materialized_views()` для обновления всех представлений
- ✅ Триггер `trigger_refresh_views_ohlcv` для автоматического обновления критических представлений
- ✅ Индексы для оптимизации запросов к представлениям

## ✅ Этап 9: Мониторинг и метрики

### 📊 Система мониторинга

**Файл:** `src/db/migrate_monitoring_metrics.py`

**Созданные таблицы:**
1. **migration_logs** - логи выполнения миграций
2. **lock_logs** - логи блокировок и их длительности
3. **performance_metrics** - метрики производительности
4. **alerts** - система алертов с уровнями важности

**Автоматические функции:**
- ✅ `collect_performance_metrics()` - сбор метрик производительности
- ✅ `monitor_locks()` - мониторинг блокировок
- ✅ `create_migration_alert()` - создание алертов о неудачных миграциях
- ✅ `export_metrics_json()` - экспорт метрик в JSON
- ✅ `log_migration_changes()` - автоматическое логирование миграций

**Prometheus-совместимые метрики:**
- ✅ `prometheus_metrics` - представление для экспорта в Prometheus
- ✅ Метрики длительности миграций
- ✅ Метрики размера таблиц
- ✅ Метрики активных подключений
- ✅ Метрики блокировок

### 🖥️ CLI для мониторинга

**Файл:** `src/db/monitoring_cli.py`

**Доступные команды:**
- `collect` - сбор метрик производительности
- `alerts` - просмотр алертов с фильтрацией по уровню важности
- `metrics` - просмотр метрик за указанный период
- `locks` - просмотр логов блокировок
- `export` - экспорт метрик в JSON файл
- `prometheus` - просмотр Prometheus-совместимых метрик
- `monitor` - запуск мониторинга блокировок
- `refresh` - обновление материализованных представлений

## 🚀 Готово к использованию

### Запуск новых миграций
```bash
python src/main_v2.py --migrations
```

### Использование CLI мониторинга
```bash
# Сбор метрик
python src/db/monitoring_cli.py collect

# Просмотр алертов
python src/db/monitoring_cli.py alerts

# Просмотр метрик
python src/db/monitoring_cli.py metrics --hours 24
```

### Использование материализованных представлений
```sql
-- Статистика по символам
SELECT * FROM mv_symbol_stats;

-- Последние цены
SELECT * FROM mv_latest_prices;

-- Топ активных символов
SELECT * FROM mv_top_symbols;
```
