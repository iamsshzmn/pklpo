## 🔧 API Reference

### Основные функции
- `compute_features()` - единый интерфейс расчёта
- `get_available_features()` - список всех индикаторов
- `get_feature_info()` - информация об индикаторе
- `get_features_by_type()` - индикаторы по типу

### Валидация
- `validate_ohlcv_data()` - валидация OHLCV данных
- `validate_feature_compatibility()` - проверка совместимости
- `validate_lookahead_safety()` - проверка безопасности

### Утилиты
- `volatility_normalize_features()` - волатильностная нормировка
- `zscore_normalize_features()` - Z-score нормализация
- `minmax_normalize_features()` - Min-max нормализация
- `calculate_feature_statistics()` - статистики по индикаторам

## 🖥️ CLI интерфейс

### Команды
- `compute` - расчёт признаков
- `validate` - валидация данных

### Примеры использования
```bash
# Расчёт индикаторов
python -m src.features.cli compute \
  --input data/ohlcv.csv \
  --specs rsi_14,atr_14,ema_12 \
  --normalize \
  --output data/features.csv

# Валидация данных
python -m src.features.cli validate \
  --input data/ohlcv.csv \
  --specs rsi_14,atr_14,ema_12
```
