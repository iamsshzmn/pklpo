# Visual Module

Модуль для визуализации торговых данных, технических индикаторов и их комбинаций с использованием интерактивных веб-интерфейсов на базе Dash и Plotly.

## Обзор

Модуль `visual` предназначен для:
- Интерактивной визуализации OHLCV данных
- Отображения технических индикаторов
- Анализа комбинаций индикаторов
- Создания веб-дашбордов для торгового анализа

## Структура модуля

### Dash Indicators Dashboard (`dash_indicators.py`)

Интерактивное веб-приложение для визуализации торговых данных:

#### Основные компоненты:
- **Символы**: выпадающий список доступных торговых пар
- **Таймфреймы**: выбор временного интервала
- **Типы данных**: переключение между OHLCV, индикаторами и комбинациями
- **Интерактивные графики**: динамическое обновление при изменении параметров

#### Поддерживаемые типы визуализации:

##### 1. OHLCV данные
- **Свечные графики**: отображение цен открытия, максимума, минимума и закрытия
- **Объем**: гистограмма торгового объема
- **Интерактивность**: масштабирование, панорамирование, наведение

##### 2. Технические индикаторы
- **Цена закрытия**: базовая линия для сравнения
- **MACD**: Moving Average Convergence Divergence
- **RSI**: Relative Strength Index
- **EMA21**: Exponential Moving Average (21 период)
- **Bollinger Bands**: верхняя и нижняя полосы

##### 3. Комбинации индикаторов
- **Value columns**: отображение значений комбинаций (value_1...value_10)
- **Динамический выбор**: автоматическое определение доступных комбинаций
- **Ограничение данных**: показ последних 500 записей для производительности

## Функциональность

### Интерактивные элементы управления:

1. **Выбор символа**
   - Динамическая загрузка из базы данных
   - Автоматическое обновление при изменении

2. **Выбор таймфрейма**
   - Поддержка различных временных интервалов
   - Синхронизация с доступными данными

3. **Переключение типов данных**
   - OHLCV: свечные графики + объем
   - Indicators: технические индикаторы
   - Combinations: комбинации индикаторов

### Графические возможности:

#### Свечные графики (OHLCV)
- Цветовое кодирование: зеленые (рост) / красные (падение) свечи
- Отображение объема в отдельной панели
- Интерактивные подсказки при наведении

#### Линейные графики (Индикаторы)
- Множественные линии на одном графике
- Автоматическое определение цветов
- Легенда с возможностью скрытия/показа линий

#### Комбинации индикаторов
- Отображение до 10 значений комбинаций
- Динамическое определение доступных колонок
- Ограничение данных для производительности

## Интеграция с базой данных

### Поддерживаемые таблицы:
- **ohlcv**: данные OHLCV (Open, High, Low, Close, Volume)
- **indicators_p**: технические индикаторы
- **indicator_combinations**: комбинации индикаторов

### Структура запросов:
```sql
-- OHLCV данные
SELECT ts, open, high, low, close, volume
FROM ohlcv
WHERE symbol = %s AND timeframe = %s
ORDER BY ts ASC

-- Индикаторы
SELECT * FROM indicators_p
WHERE symbol = %s AND timeframe = %s
ORDER BY timestamp ASC

-- Комбинации
SELECT * FROM indicator_combinations
WHERE symbol = %s AND timeframe = %s
ORDER BY ts ASC
LIMIT 500
```
